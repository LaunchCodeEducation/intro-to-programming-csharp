.. _pow-examples:

``Pow()`` Examples
====================================

Returns value of a number raised to specified power.

**General Syntax**

.. sourcecode:: csharp

   Math.Pow(number, power)


.. admonition:: Example

   .. sourcecode:: csharp
      :linenos:

      Console.WriteLine(Math.Pow(4,2));
      Console.WriteLine(Math.Pow(23.4,8));

   **Console Output**

   :: 

      16
      89893203860.5205
