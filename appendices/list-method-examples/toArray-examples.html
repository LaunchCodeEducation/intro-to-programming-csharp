ToArray Method¶

Copies the elements within the list into an array.

This does not create the array, like ToCharArray. You must declare the array of the same size as your list.

// Option A
string[] listArray = new string[listName.Count];
listArray = listName.ToArray();

// Option B
string[] listArray = listName.ToArray();

Example

 1List<string> planets = new List<string>();
 2   planets.Add("Mercury");
 3   planets.Add("Venus");
 4   planets.Add("Earth");
 5   planets.Add("Mars");
 6   planets.Add("Jupiter");
 7   planets.Add("Saturn");
 8   planets.Add("Uranus");
 9   planets.Add("Neptune");
10
11string[] planetArray = planets.ToArray();
12
13foreach(string planet in planetArray)
14{
15   Console.WriteLine(planet);
16}
