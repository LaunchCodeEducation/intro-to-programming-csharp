GetValue Examples¶

The general syntax for this method is:

arrayName.GetValue(index)

This method returns the value of the specified index.

Example

 1string[] pets = new string[]
 2{
 3   "Alyce",
 4   "Thisbe",
 5   "Francis",
 6   "Scratch",
 7   "Beatrice",
 8   "Shadow",
 9   "Beatrice"
10};
11
12Console.WriteLine(pets.GetValue(2));
13Console.WriteLine(pets.GetValue(8));

Output

Francis
Unhandled Expectation: System.IndexOutOfRangeException
